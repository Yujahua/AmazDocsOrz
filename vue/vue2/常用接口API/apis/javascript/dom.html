属性 / 方法 描述
document.getElementById() 返回对拥有指定 id 的第一个对象的引用
document.getElementsByTagName() 返回带有指定标签名的对象集合
document.getElementsByName() 返回带有指定名称的对象集合
属性 / 方法 描述
element.getAttribute() 返回指定元素的属性值
element.setAttribute() 设置或者改变指定属性并指定值
element.innerHTML 设置或者返回元素的html内容
element.innerText 设置或者返回元素的文本内容
属性 / 方法 描述
element.appendChild() 元素内部的结尾插入新结点，新结点由第一个参数指定
element.insertBefore() 元素内部第二个参数指定的孩子结点前插入新结点
element.removeChild() 删除元素的一个子元素，子元素由第一个参数指定
element.replaceChild(newnode,oldnode) 替换一个子元素
属性 / 方法 描述
document.defaultView.getComputedStyle() 非IE浏览器获取全部样式，第一个参数是结点，第二个参数是伪类，不使用设置null，具体样式使用getPropertyValue(key)获取
element.currentStyle IE浏览器获取全部样式，具体样式使用getPropertyValue(key)获取
element.style 设置或返回元素的样式属性(和上面二个方法对比，返回的是计算前的样式)
